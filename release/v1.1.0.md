# v1.1.0

Documentation refinement release for `unit-converter`.

## Highlights

- Updated the MkDocs Material theme to use an indigo primary color for a
  slightly blue-purple navigation bar.
- Set the documentation fonts to Roboto and Roboto Mono.
- Refined the documentation home page by replacing the data-set summary table
  with a minimal `convert()` example.

## Notes

- This release focuses on the GitHub Pages documentation experience.
- Package conversion behavior is unchanged from v1.0.0.
