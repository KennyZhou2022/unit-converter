# v1.0.0

First GitHub release candidate for `unit-converter`.

## Highlights

- Added NIST SP 811 Appendix B.9 conversion data.
- Added the `convert(value, from_unit, to_unit)` public API.
- Added conversion through connected intermediate units.
- Added bundled JSON runtime data and supported-unit catalog.
- Added explicit temperature and temperature-interval unit labels.
- Added MkDocs documentation with GitHub Pages deployment workflow.
- Added pytest coverage for core conversion behavior and bundled data.

## Notes

- Runtime conversion uses bundled JSON data; the source PDF is not required at
  runtime.
- The source PDF is not intended to be redistributed from package data.
